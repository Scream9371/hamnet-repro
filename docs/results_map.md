# Results Map

This document records how paper tables and figures map to the reference result files in this repository.

## Reference Summary Files

- `results_reference/samplewise_summary.json`
- `results_reference/cpdp_summary.json`
- `results_reference/ablation_summary.json`
- `results_reference/interpretability_summary.json`

## Intended Mapping

The repository is designed so that:

1. The sample-wise main table maps to `samplewise_summary.json`.
2. The fixed CPDP main table maps to `cpdp_summary.json`.
3. The structural ablation table maps to `ablation_summary.json`.
4. The interpretability table and related plots map to `interpretability_summary.json`.
